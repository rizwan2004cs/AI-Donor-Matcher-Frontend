> **Branch:** `feat/ngo-profile-completion`

> **Route:** `/dashboard/ngo/complete-profile`
> **Depends on:** Axios, AuthContext, ProtectedRoute (NGO)

---

## Overview

A guided profile completion screen for newly registered NGOs. Shows a progress bar, required-field checklist with green checkmarks, photo upload, and geolocation detection. NGOs are only visible on the donor map once all required fields are complete.

## API Contract

### Fetch Current Profile

```
GET /api/ngo/my/profile

Response (200):
{
  "name": "Hope Foundation",
  "address": "",
  "contactEmail": "hope@ngo.org",
  "contactPhone": "",
  "description": "",
  "category": "",
  "photoUrl": null,
  "profileComplete": false
}
```

### Save Profile

```
PUT /api/ngo/my/profile
Content-Type: application/json

Request Body:
{
  "name": "Hope Foundation",
  "address": "12 Gandhi Road, Chennai",
  "contactEmail": "hope@ngo.org",
  "contactPhone": "+91 9876543210",
  "description": "We provide food and clothing to underprivileged communities in Chennai.",
  "category": "FOOD"
}

Response (200):
{ "message": "Profile updated", "profileComplete": true }
```

### Upload Photo

```
POST /api/ngo/my/photo
Content-Type: multipart/form-data

Body: photo (file — JPG/PNG, max 2MB)

Response (200):
{ "photoUrl": "https://res.cloudinary.com/..." }
```

## Component Specification

### State

```js
const navigate = useNavigate();
const { user, login } = useAuth();

const [profile, setProfile] = useState({
  name: "",
  address: "",
  contactEmail: "",
  contactPhone: "",
  description: "",
  category: "",
});
const [photoUrl, setPhotoUrl] = useState(null);
const [loading, setLoading] = useState(true);
const [saving, setSaving] = useState(false);
const [error, setError] = useState(null);
```

### Required Fields & Progress

```js
const REQUIRED_FIELDS = [
  { key: "name", label: "Organisation Name" },
  { key: "address", label: "Full Address" },
  { key: "contactEmail", label: "Contact Email" },
  { key: "contactPhone", label: "Contact Phone" },
  { key: "description", label: "Organisation Description (min 50 chars)" },
  { key: "category", label: "Category of Work" },
];

const isFilled = (key) => {
  const val = profile[key]?.toString().trim();
  if (!val) return false;
  if (key === "description" && val.length < 50) return false;
  return true;
};

const filledCount = REQUIRED_FIELDS.filter((f) => isFilled(f.key)).length;
const progressPct = Math.round((filledCount / REQUIRED_FIELDS.length) * 100);
const allComplete = filledCount === REQUIRED_FIELDS.length;
```

### Photo Upload Handler

```js
const handlePhotoUpload = async (e) => {
  const file = e.target.files[0];
  if (!file) return;

  // Client-side validation
  if (!["image/jpeg", "image/png"].includes(file.type)) {
    setError("Only JPG and PNG files are allowed");
    return;
  }
  if (file.size > 2 * 1024 * 1024) {
    setError("File must be smaller than 2MB");
    return;
  }

  const formData = new FormData();
  formData.append("photo", file);

  try {
    const res = await api.post("/api/ngo/my/photo", formData, {
      headers: { "Content-Type": "multipart/form-data" },
    });
    setPhotoUrl(res.data.photoUrl);
  } catch (err) {
    setError("Failed to upload photo");
  }
};
```

### Save Handler

```js
const handleSave = async () => {
  if (!allComplete) return;
  setSaving(true);
  setError(null);

  try {
    const res = await api.put("/api/ngo/my/profile", profile);
    // Update AuthContext so profileComplete flag is true
    login({ ...user, profileComplete: true }, localStorage.getItem("token"));
    navigate("/dashboard/ngo");
  } catch (err) {
    setError(err.response?.data?.message || "Failed to save profile");
  } finally {
    setSaving(false);
  }
};
```

### Geolocation for Address

```js
const detectLocation = () => {
  navigator.geolocation.getCurrentPosition(
    async (pos) => {
      const { latitude, longitude } = pos.coords;
      // Optionally reverse-geocode via a free API
      // For now, store coordinates as placeholder
      setProfile((prev) => ({
        ...prev,
        address: `Lat: ${latitude.toFixed(4)}, Lng: ${longitude.toFixed(4)}`,
      }));
    },
    () => alert("Location access denied")
  );
};
```

## Behaviour Notes

- NGO is only visible on the donor map once `profileComplete === true` (backend enforced)
- If NGO logs in and `profileComplete === false`, Login (1.4) redirects here
- Description must be at least 50 characters
- Photo upload is optional but recommended
- "Detect Location" uses `navigator.geolocation` to auto-fill address
- Category dropdown: Food / Clothing / Medicine / Education / Household / Other
- On save success, update AuthContext user so the app no longer redirects here
