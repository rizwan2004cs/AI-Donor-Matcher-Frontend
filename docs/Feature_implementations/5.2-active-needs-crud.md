## Overview

The "Active Needs" section on the NGO Dashboard. Lists current needs with progress bars. Supports Add (max 5), Edit, Delete (only when no active pledges), and a lock icon when pledges exist.

## API Contract

### Fetch NGO's Needs

```
GET /api/ngo/my/needs

Response (200):
[
  {
    "id": 101,
    "category": "FOOD",
    "itemName": "Rice (10kg bags)",
    "urgency": "URGENT",
    "quantityRequired": 50,
    "quantityPledged": 35,
    "expiryDate": "2026-04-01",
    "status": "ACTIVE"
  }
]
```

### Add Need

```
POST /api/needs
Content-Type: application/json

Request Body:
{
  "category": "FOOD",
  "itemName": "Rice (10kg bags)",
  "description": "10kg bags of basmati rice",
  "quantityRequired": 50,
  "urgency": "URGENT",
  "expiryDate": "2026-04-01"
}

Response (201):
{ "id": 102, ... }
```

### Edit Need

```
PUT /api/needs/{id}
Content-Type: application/json

Request Body:
{ "category", "itemName", "description", "quantityRequired", "urgency", "expiryDate" }

Response (200):
{ "message": "Need updated" }
```

### Delete Need

```
DELETE /api/needs/{id}

Response (200):
{ "message": "Need deleted" }
```

### Add Need Modal

```jsx
// src/components/AddNeedModal.jsx
export default function AddNeedModal({ onClose, onSave, initialData }) {
  const [form, setForm] = useState(initialData || {
    category: "FOOD",
    itemName: "",
    description: "",
    quantityRequired: 1,
    urgency: "NORMAL",
    expiryDate: "",
  });
  const [saving, setSaving] = useState(false);

  const handleSubmit = async () => {
    setSaving(true);
    await onSave(form);
    setSaving(false);
    onClose();
  };

  return (
    <div className="fixed inset-0 bg-black/50 flex items-center justify-center z-50">
      <div className="bg-white rounded-lg p-6 w-96">
        <h3 className="text-lg font-semibold mb-4">
          {initialData ? "Edit Need" : "Add New Need"}
        </h3>

        {/* Category */}
        <select value={form.category} onChange={(e) => setForm({...form, category: e.target.value})}
          className="w-full border rounded px-3 py-2 mb-3 text-sm">
          <option value="FOOD">Food</option>
          <option value="CLOTHING">Clothing</option>
          <option value="MEDICINE">Medicine</option>
          <option value="EDUCATION">Education</option>
          <option value="HOUSEHOLD">Household</option>
          <option value="OTHER">Other</option>
        </select>

        {/* Item Name */}
        <input type="text" placeholder="Item name *" value={form.itemName}
          onChange={(e) => setForm({...form, itemName: e.target.value})}
          className="w-full border rounded px-3 py-2 mb-3 text-sm" required />

        {/* Description */}
        <textarea placeholder="Description (optional)" value={form.description}
          onChange={(e) => setForm({...form, description: e.target.value})}
          className="w-full border rounded px-3 py-2 mb-3 text-sm" rows={3} />

        {/* Quantity */}
        <input type="number" min={1} placeholder="Quantity *" value={form.quantityRequired}
          onChange={(e) => setForm({...form, quantityRequired: parseInt(e.target.value) || 1})}
          className="w-full border rounded px-3 py-2 mb-3 text-sm" />

        {/* Urgency */}
        <div className="flex gap-4 mb-3">
          <label className="flex items-center gap-1 text-sm">
            <input type="radio" value="NORMAL" checked={form.urgency === "NORMAL"}
              onChange={() => setForm({...form, urgency: "NORMAL"})} /> Normal
          </label>
          <label className="flex items-center gap-1 text-sm">
            <input type="radio" value="URGENT" checked={form.urgency === "URGENT"}
              onChange={() => setForm({...form, urgency: "URGENT"})} /> Urgent
          </label>
        </div>

        {/* Expiry Date */}
        <input type="date" value={form.expiryDate}
          onChange={(e) => setForm({...form, expiryDate: e.target.value})}
          className="w-full border rounded px-3 py-2 mb-4 text-sm" />

        <div className="flex gap-2">
          <button onClick={handleSubmit} disabled={!form.itemName || saving}
            className="bg-indigo-600 text-white px-4 py-2 rounded text-sm disabled:opacity-50">
            {saving ? "Saving..." : "Save"}
          </button>
          <button onClick={onClose} className="bg-gray-200 px-4 py-2 rounded text-sm">
            Cancel
          </button>
        </div>
      </div>
    </div>
  );
}
```

## Acceptance Criteria

- [ ] Each need shows: category dot, item name, urgency, progress bar, remaining
- [ ] Needs with `quantityPledged > 0` show 🔒 lock icon
- [ ] Locked needs show only "Mark as Fulfilled" button
- [ ] Unlocked needs show "Edit" and "Delete" buttons
- [ ] "+ Add New Need" opens modal
- [ ] Add button disabled when 5 needs exist (with message)
- [ ] Add modal has all fields: category, item, description, quantity, urgency, expiry
- [ ] Edit modal pre-fills existing need data
- [ ] Delete sends DELETE request and removes card
- [ ] POST/PUT/DELETE update the list immediately
