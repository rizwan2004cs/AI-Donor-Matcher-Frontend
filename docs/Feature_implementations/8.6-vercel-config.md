# 8.6 Vercel Config

Last updated: 2026-03-31

## Production Frontend

- `https://ai-donor-matcher-frontend.vercel.app`

## Required Env Vars

- `VITE_API_BASE_URL=https://ai-donor-matcher-backend.onrender.com`
- Firebase web config vars
- `VITE_OSRM_URL`
- optional `VITE_NOMINATIM_URL`

## Cross-system Notes

- backend CORS must allow the Vercel frontend domain
- Firebase authorized domains must include `ai-donor-matcher-frontend.vercel.app`
