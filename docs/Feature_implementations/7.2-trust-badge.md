## Overview

A reusable badge component that visually represents an NGO's trust score. Displays a star icon, numeric score, and tier label with colour-coded styling. Used on NGO cards, profile pages, and admin tables.

## Trust Tiers

| Tier | Score Range | Colour | Tailwind Classes |
|------|-------------|--------|------------------|
| New | 0 – 39 | Gray | `bg-gray-100 text-gray-600` |
| Established | 40 – 69 | Yellow | `bg-yellow-100 text-yellow-700` |
| Trusted | 70 – 100 | Green | `bg-green-100 text-green-700` |

---

## Component Specification

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `score` | `number` | `0` | Value 0–100 |

### Full Component

```jsx
import { Star } from "lucide-react";

const tiers = [
  { max: 39,  label: "New",         bg: "bg-gray-100",   text: "text-gray-600" },
  { max: 69,  label: "Established", bg: "bg-yellow-100", text: "text-yellow-700" },
  { max: 100, label: "Trusted",     bg: "bg-green-100",  text: "text-green-700" },
];

function getTier(score) {
  return tiers.find((t) => score <= t.max) || tiers[tiers.length - 1];
}

export default function TrustBadge({ score = 0 }) {
  const tier = getTier(score);

  return (
    <span
      className={`inline-flex items-center gap-1 px-2 py-0.5 rounded-full text-xs font-medium ${tier.bg} ${tier.text}`}
    >
      <Star size={12} />
      {score} — {tier.label}
    </span>
  );
}
```

## Behaviour Notes

- Score is clamped by backend (0–100). Component trusts the value.
- If `score` is `null` or `undefined`, defaults to `0` (tier "New").
- Icon uses `lucide-react` `Star` at 12 px to stay compact.
