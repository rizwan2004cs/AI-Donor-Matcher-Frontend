# Feature 2.4 — Auto-Expand Nationwide

> **Branch:** `feat/auto-expand`
> **File:** `src/pages/DiscoveryMap.jsx` (logic addition)
> **Depends on:** DiscoveryMap (2.1)

---

## Overview

If the NGO search API returns an empty result set for the user's location and radius, automatically re-fetch without the radius parameter to get nationwide results. Display a yellow info banner to inform the user.

## Implementation

### State

```js
const [expanded, setExpanded] = useState(false);
```

### Logic (inside the fetch function in DiscoveryMap)

```js
const fetchNgos = async () => {
  setLoading(true);
  setExpanded(false);
  setError(null);

  try {
    const params = { lat: userLat, lng: userLng, radius: filters.radius };
    if (filters.category) params.category = filters.category;
    if (filters.search) params.search = filters.search;

    const res = await api.get("/api/ngos", { params });

    if (res.data.length === 0) {
      // Remove radius and re-fetch for nationwide results
      const { radius, ...nationwideParams } = params;
      const fallback = await api.get("/api/ngos", { params: nationwideParams });
      setNgos(fallback.data);
      setExpanded(true);
    } else {
      setNgos(res.data);
    }
  } catch (err) {
    setError(err.response?.data?.message || "Failed to load NGOs");
  } finally {
    setLoading(false);
  }
};
```

### Banner JSX

```jsx
{expanded && (
  <div className="bg-yellow-100 text-yellow-800 text-sm px-4 py-2 text-center border-b border-yellow-200">
    ⚠️ No NGOs found nearby. Showing all available NGOs.
  </div>
)}
```

Place this between the FilterBar and the map/side-list container.

---

## Behaviour Notes

- The second API call omits **only** the `radius` parameter — `category` and `search` filters still apply
- If the nationwide call also returns empty, show the normal "No NGOs found" empty state
- When the user changes filters (e.g., adjusts radius), `expanded` resets to `false` and the normal flow retries
- The banner is dismissible (user can close it) — optional enhancement
