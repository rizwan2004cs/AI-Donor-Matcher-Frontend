# Feature 6.5 — Suspend NGO (Cascade)

> **Branch:** `feat/admin-suspend`
> **File:** `src/pages/AdminDashboard.jsx`
> **Depends on:** Axios, Admin Management (6.4), Report Queue (6.3)

## Overview

An atomic admin action that suspends an NGO. The backend cascades: closes all active needs, cancels all pledges, notifies all affected donors, and removes the NGO from the discovery map.

## API Contract

```
POST /api/admin/ngos/{id}/suspend

Response (200):
{
  "message": "NGO suspended",
  "needsClosed": 3,
  "pledgesCancelled": 8,
  "donorsNotified": 6
}
```

## Component Specification

### Confirmation Modal

```jsx
const [suspendTarget, setSuspendTarget] = useState(null);
const [suspending, setSuspending] = useState(false);
const [suspendResult, setSuspendResult] = useState(null);

const handleSuspend = (ngoId) => {
  setSuspendTarget(ngoId);
};

const confirmSuspend = async () => {
  setSuspending(true);
  try {
    const res = await api.post(`/api/admin/ngos/${suspendTarget}/suspend`);
    setSuspendResult(res.data);

    // Remove from NGO list
    setNgos((prev) => prev.filter((n) => n.id !== suspendTarget));
    // Remove from reports list
    setReports((prev) => prev.filter((r) => r.ngoId !== suspendTarget));
  } catch (err) {
    alert(err.response?.data?.message || "Failed to suspend NGO");
  } finally {
    setSuspending(false);
  }
};
```

## Backend Cascade (for reference)

When `POST /api/admin/ngos/{id}/suspend` is called:

1. Set NGO status to `SUSPENDED`
2. Close all active needs (set status to `CLOSED`)
3. Cancel all active pledges (set status to `CANCELLED`)
4. Send notification emails to all donors with active pledges
5. Remove NGO from map query results
6. Return count of affected entities
