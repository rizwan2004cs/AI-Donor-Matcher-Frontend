## Overview

A horizontal filter strip at the top of the Discovery Map page. Contains an NGO name search input (debounced 300 ms), a category dropdown, and a radius slider (1–50 km). All filters drive the same API call.

### Full Component

```jsx
export default function FilterBar({ filters, setFilters }) {
  const [searchInput, setSearchInput] = useState(filters.search);

  // Debounce search by 300ms
  useEffect(() => {
    const timer = setTimeout(() => {
      setFilters((prev) => ({ ...prev, search: searchInput }));
    }, 300);
    return () => clearTimeout(timer);
  }, [searchInput]);

  const categories = [
    { value: "", label: "All Categories" },
    { value: "FOOD", label: "Food" },
    { value: "CLOTHING", label: "Clothing" },
    { value: "MEDICINE", label: "Medicine" },
    { value: "EDUCATION", label: "Education" },
    { value: "HOUSEHOLD", label: "Household" },
    { value: "OTHER", label: "Other" },
  ];

  return (
    <div className="bg-white border-b px-4 py-3 flex items-center gap-4 flex-wrap">
      {/* Search */}
      <input
        type="text"
        placeholder="Search NGO name..."
        value={searchInput}
        onChange={(e) => setSearchInput(e.target.value)}
        className="border rounded px-3 py-2 text-sm w-60"
      />

      {/* Category dropdown */}
      <select
        value={filters.category}
        onChange={(e) =>
          setFilters((prev) => ({ ...prev, category: e.target.value }))
        }
        className="border rounded px-3 py-2 text-sm"
      >
        {categories.map((c) => (
          <option key={c.value} value={c.value}>
            {c.label}
          </option>
        ))}
      </select>

      {/* Radius slider */}
      <div className="flex items-center gap-2">
        <label className="text-sm text-gray-600">Radius:</label>
        <input
          type="range"
          min={1}
          max={50}
          value={filters.radius}
          onChange={(e) =>
            setFilters((prev) => ({
              ...prev,
              radius: parseInt(e.target.value),
            }))
          }
          className="w-32"
        />
        <span className="text-sm font-medium w-12">{filters.radius} km</span>
      </div>
    </div>
  );
}
```

## API Integration

All three filters feed into the same `GET /api/ngos` call in DiscoveryMap:

```
GET /api/ngos?lat=13.08&lng=80.27&radius=10&category=FOOD&search=hope
```

- `search` → filters by NGO name (partial match, backend handles)
- `category` → filters by NGO's top need category
- `radius` → limits results to within N km

---

## Behaviour Notes

- **Debounce:** Search input waits 300 ms after typing stops before triggering API call
- **Category + Radius:** Changes trigger immediately (no debounce needed for selects/sliders)
- **Empty search:** Omit `search` param from API call
- **Empty category:** Omit `category` param (show all categories)
