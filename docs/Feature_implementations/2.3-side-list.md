## Overview

A scrollable right-side panel on the Discovery Map page listing NGOs sorted by distance (ascending). Within 2 km bands, urgent needs sort first. Each card shows name, distance, urgency badge, top need item, and remaining count.

### Sorting Logic

```js
const sortedNgos = [...ngos].sort((a, b) => {
  // Sort by 2km bands first
  const bandA = Math.floor(a.distanceKm / 2);
  const bandB = Math.floor(b.distanceKm / 2);

  if (bandA !== bandB) return bandA - bandB;

  // Within same band, urgent first
  const urgencyOrder = { URGENT: 0, NORMAL: 1 };
  const urgA = urgencyOrder[a.topNeedUrgency] ?? 1;
  const urgB = urgencyOrder[b.topNeedUrgency] ?? 1;
  if (urgA !== urgB) return urgA - urgB;

  // Tie-break by distance
  return a.distanceKm - b.distanceKm;
});
```

### Full Component

```jsx
import { useNavigate } from "react-router-dom";
import { CATEGORY_COLORS } from "../utils/categoryColors";

export default function SideList({ ngos }) {
  const navigate = useNavigate();

  const sortedNgos = [...ngos].sort((a, b) => {
    const bandA = Math.floor(a.distanceKm / 2);
    const bandB = Math.floor(b.distanceKm / 2);
    if (bandA !== bandB) return bandA - bandB;
    const urgA = a.topNeedUrgency === "URGENT" ? 0 : 1;
    const urgB = b.topNeedUrgency === "URGENT" ? 0 : 1;
    if (urgA !== urgB) return urgA - urgB;
    return a.distanceKm - b.distanceKm;
  });

  return (
    <div className="w-80 bg-white border-l overflow-y-auto">
      <h3 className="px-4 py-3 font-semibold text-sm border-b">
        Nearby NGOs ({ngos.length})
      </h3>

      {sortedNgos.map((ngo) => (
        <div
          key={ngo.id}
          onClick={() => navigate(`/ngo/${ngo.id}`)}
          className="px-4 py-3 border-b hover:bg-gray-50 cursor-pointer"
        >
          <div className="flex items-center justify-between">
            <span className="font-medium text-sm">{ngo.name}</span>
            <span className="text-xs text-gray-400">
              {ngo.distanceKm.toFixed(1)} km
            </span>
          </div>

          <div className="flex items-center gap-2 mt-1">
            {/* Category colour dot */}
            <span
              className="w-2.5 h-2.5 rounded-full inline-block"
              style={{
                backgroundColor:
                  CATEGORY_COLORS[ngo.pinCategory] || "#6B7280",
              }}
            />
            <span className="text-xs text-gray-600">
              {ngo.topNeedItem} — {ngo.topNeedRemaining} remaining
            </span>
          </div>

          {ngo.topNeedUrgency === "URGENT" && (
            <span className="inline-block mt-1 text-xs bg-red-100 text-red-700 px-2 py-0.5 rounded-full">
              Urgent
            </span>
          )}
        </div>
      ))}

      {ngos.length === 0 && (
        <p className="px-4 py-6 text-sm text-gray-400 text-center">
          No NGOs found.
        </p>
      )}
    </div>
  );
}
```

## Behaviour Notes

- Clicking a card navigates to `/ngo/:id` (full profile page)
- List scrolls independently of the map
- Urgent items get a red badge
- Category dot colour matches the map pin colour
- Empty state shows "No NGOs found" message
