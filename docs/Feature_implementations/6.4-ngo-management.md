## API Contract

### Fetch All Verified NGOs

```
GET /api/admin/ngos

Response (200):
[
  {
    "id": 1,
    "name": "Hope Foundation",
    "trustScore": 78,
    "trustLabel": "Established",
    "email": "hope@ngo.org",
    "activeNeedsCount": 3,
    "status": "ACTIVE"
  }
]
```

### View NGO's Needs (inline expand)

```
GET /api/admin/ngos/{id}/needs

Response (200):
[
  {
    "id": 101,
    "category": "FOOD",
    "itemName": "Rice (10kg bags)",
    "quantityRequired": 50,
    "quantityPledged": 35,
    "status": "ACTIVE"
  }
]
```

### Edit Need (admin override)

```
PUT /api/admin/needs/{id}
Content-Type: application/json

Request Body:
{ "category", "itemName", "description", "quantityRequired", "urgency", "expiryDate" }

Response (200):
{ "message": "Need updated" }
```

### Delete Need (admin override — cascades pledge cancellation)

```
DELETE /api/admin/needs/{id}

Response (200):
{ "message": "Need deleted, pledges cancelled" }
```

### Suspend NGO

```
POST /api/admin/ngos/{id}/suspend

Response (200):
{ "message": "NGO suspended" }
```

## Component Specification

### State

```js
const [ngos, setNgos] = useState([]);
const [expandedNgo, setExpandedNgo] = useState(null);
const [ngoNeeds, setNgoNeeds] = useState([]); // needs for expanded NGO
```

### Fetch

```js
useEffect(() => {
  api.get("/api/admin/ngos")
    .then((res) => setNgos(res.data))
    .catch(() => {});
}, []);
```

### Expand to View Needs

```js
const handleViewNeeds = async (ngoId) => {
  if (expandedNgo === ngoId) {
    setExpandedNgo(null);
    return;
  }
  const res = await api.get(`/api/admin/ngos/${ngoId}/needs`);
  setNgoNeeds(res.data);
  setExpandedNgo(ngoId);
};
```

### Layout JSX

```jsx
<div className="mt-8">
  <h2 className="text-lg font-semibold mb-3">NGO Management</h2>

  <table className="w-full text-sm">
    <thead>
      <tr className="border-b text-left text-gray-500">
        <th className="py-2">Name</th>
        <th>Trust</th>
        <th>Needs</th>
        <th>Actions</th>
      </tr>
    </thead>
    <tbody>
      {ngos.map((ngo) => (
        <>
          <tr key={ngo.id} className="border-b">
            <td className="py-3 font-medium">{ngo.name}</td>
            <td><TrustBadge score={ngo.trustScore} label={ngo.trustLabel} /></td>
            <td>{ngo.activeNeedsCount}</td>
            <td className="flex gap-2 py-3">
              <button
                onClick={() => handleViewNeeds(ngo.id)}
                className="text-blue-600 text-sm underline"
              >
                {expandedNgo === ngo.id ? "Hide" : "View Needs"}
              </button>
              <button
                onClick={() => handleSuspend(ngo.id)}
                className="text-red-600 text-sm underline"
              >
                Suspend
              </button>
            </td>
          </tr>

          {/* Inline expanded needs */}
          {expandedNgo === ngo.id && (
            <tr>
              <td colSpan={4} className="bg-gray-50 p-4">
                {ngoNeeds.length === 0 ? (
                  <p className="text-gray-400 text-sm">No active needs.</p>
                ) : (
                  ngoNeeds.map((need) => (
                    <div key={need.id} className="flex items-center justify-between py-2 border-b last:border-0">
                      <div className="flex items-center gap-2">
                        <span
                          className="w-2.5 h-2.5 rounded-full"
                          style={{ backgroundColor: CATEGORY_COLORS[need.category] }}
                        />
                        <span>{need.itemName}</span>
                        <span className="text-gray-400">
                          {need.quantityPledged}/{need.quantityRequired}
                        </span>
                      </div>
                      <div className="flex gap-2">
                        <button className="text-sm text-blue-600 underline">Edit</button>
                        <button
                          onClick={() => handleDeleteNeed(need.id)}
                          className="text-sm text-red-600 underline"
                        >
                          Delete
                        </button>
                      </div>
                    </div>
                  ))
                )}
              </td>
            </tr>
          )}
        </>
      ))}
    </tbody>
  </table>
</div>
```
