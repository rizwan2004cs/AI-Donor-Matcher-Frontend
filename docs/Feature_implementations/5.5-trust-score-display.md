## Overview

The NGO Dashboard header shows the NGO's name, trust score (numeric value), tier label (New / Established / Trusted), active needs count, and total fulfilled count. Updates automatically after a fulfillment action.

## API Contract

Trust data comes as part of the profile fetch or as a response from fulfillment:

```
GET /api/ngo/my/profile

Response (partial):
{
  "name": "Hope Foundation",
  "trustScore": 78,
  "trustLabel": "Established",
  "activeNeedsCount": 3,
  "fulfilledCount": 12
}
```

### Trust Tier Thresholds (for reference)

| Tier | Score Range | Colour |
|------|------------|--------|
| New | 0–40 | Gray |
| Established | 41–70 | Yellow |
| Trusted | 71–100 | Green |

## Component Specification

### State

```js
const [ngoName, setNgoName] = useState("");
const [trustScore, setTrustScore] = useState(0);
const [trustLabel, setTrustLabel] = useState("New");
const [activeNeedsCount, setActiveNeedsCount] = useState(0);
const [fulfilledCount, setFulfilledCount] = useState(0);
```

### Fetch on Mount

```js
useEffect(() => {
  api.get("/api/ngo/my/profile").then((res) => {
    setNgoName(res.data.name);
    setTrustScore(res.data.trustScore);
    setTrustLabel(res.data.trustLabel);
    setActiveNeedsCount(res.data.activeNeedsCount);
    setFulfilledCount(res.data.fulfilledCount);
  });
}, []);
```

### Header JSX

```jsx
import TrustBadge from "../components/TrustBadge";

<div className="bg-white border-b px-6 py-4">
  <div className="flex items-center justify-between">
    <div>
      <h1 className="text-2xl font-bold">{ngoName}</h1>
      <TrustBadge score={trustScore} label={trustLabel} />
    </div>

    <div className="flex gap-6 text-center">
      <div>
        <p className="text-2xl font-bold">{activeNeedsCount}/5</p>
        <p className="text-xs text-gray-500">Active Needs</p>
      </div>
      <div>
        <p className="text-2xl font-bold">{fulfilledCount}</p>
        <p className="text-xs text-gray-500">Fulfilled</p>
      </div>
    </div>
  </div>
</div>
```

## Auto-Update After Fulfillment

When `handleFulfill()` (5.4) succeeds, the response includes `newTrustScore` and `newTrustLabel`. Update state directly:

```js
setTrustScore(res.data.newTrustScore);
setTrustLabel(res.data.newTrustLabel);
setFulfilledCount((prev) => prev + 1);
setActiveNeedsCount((prev) => prev - 1);
```
