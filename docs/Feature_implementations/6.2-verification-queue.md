## Overview

A list of NGOs with `PENDING` status awaiting admin verification. Each row shows the NGO name, submission time, a link to review uploaded documents, and Approve/Reject action buttons. Rejection requires a written reason.

## API Contract

### Fetch Pending NGOs

```
GET /api/admin/ngos/pending

Response (200):
[
  {
    "id": 5,
    "name": "New Hope Trust",
    "email": "newhope@ngo.org",
    "submittedAt": "2026-03-04T08:00:00Z",
    "documentUrl": "https://storage.example.com/docs/newhope.pdf"
  }
]
```

### Approve

```
POST /api/admin/ngos/{id}/approve

Response (200):
{ "message": "NGO approved" }
```

### Reject

```
POST /api/admin/ngos/{id}/reject
Content-Type: application/json

Request Body:
{ "reason": "Documents are expired. Please re-upload current registration." }

Response (200):
{ "message": "NGO rejected" }
```

## Component Specification

### State

```js
const [pendingNgos, setPendingNgos] = useState([]);
const [rejectTarget, setRejectTarget] = useState(null); // ngo id for reject modal
const [rejectReason, setRejectReason] = useState("");
```

### Fetch

```js
useEffect(() => {
  api.get("/api/admin/ngos/pending")
    .then((res) => setPendingNgos(res.data))
    .catch(() => {});
}, []);
```

### Approve Handler

```js
const handleApprove = async (id) => {
  try {
    await api.post(`/api/admin/ngos/${id}/approve`);
    setPendingNgos((prev) => prev.filter((n) => n.id !== id));
  } catch (err) {
    alert(err.response?.data?.message || "Failed to approve");
  }
};
```

### Reject Handler

```js
const handleReject = async () => {
  if (!rejectReason.trim()) return;
  try {
    await api.post(`/api/admin/ngos/${rejectTarget}/reject`, { reason: rejectReason });
    setPendingNgos((prev) => prev.filter((n) => n.id !== rejectTarget));
    setRejectTarget(null);
    setRejectReason("");
  } catch (err) {
    alert(err.response?.data?.message || "Failed to reject");
  }
};
```

### Layout JSX

```jsx
<div className="mt-8">
  <h2 className="text-lg font-semibold mb-3">Verification Queue ({pendingNgos.length})</h2>

  {pendingNgos.length === 0 && (
    <p className="text-gray-400 text-sm">No pending verifications.</p>
  )}

  {pendingNgos.map((ngo) => (
    <div key={ngo.id} className="border rounded p-4 mb-3 flex items-center justify-between">
      <div>
        <p className="font-medium">{ngo.name}</p>
        <p className="text-sm text-gray-500">{ngo.email} · Submitted {timeAgo(ngo.submittedAt)}</p>
      </div>
      <div className="flex items-center gap-2">
        <a
          href={ngo.documentUrl}
          target="_blank"
          rel="noopener noreferrer"
          className="text-blue-600 text-sm underline"
        >
          Review Docs
        </a>
        <button
          onClick={() => handleApprove(ngo.id)}
          className="bg-green-600 text-white px-3 py-1.5 rounded text-sm"
        >
          Approve
        </button>
        <button
          onClick={() => setRejectTarget(ngo.id)}
          className="bg-red-100 text-red-600 px-3 py-1.5 rounded text-sm"
        >
          Reject
        </button>
      </div>
    </div>
  ))}
</div>

{/* Reject Reason Modal */}
{rejectTarget && (
  <div className="fixed inset-0 bg-black/50 flex items-center justify-center z-50">
    <div className="bg-white rounded-lg p-6 w-96">
      <h3 className="font-semibold mb-3">Rejection Reason</h3>
      <textarea
        value={rejectReason}
        onChange={(e) => setRejectReason(e.target.value)}
        placeholder="Enter reason for rejection..."
        className="w-full border rounded p-3 text-sm mb-4"
        rows={4}
      />
      <div className="flex gap-2">
        <button
          onClick={handleReject}
          disabled={!rejectReason.trim()}
          className="bg-red-600 text-white px-4 py-2 rounded text-sm disabled:opacity-50"
        >
          Confirm Reject
        </button>
        <button
          onClick={() => { setRejectTarget(null); setRejectReason(""); }}
          className="bg-gray-200 px-4 py-2 rounded text-sm"
        >
          Cancel
        </button>
      </div>
    </div>
  </div>
)}
```
