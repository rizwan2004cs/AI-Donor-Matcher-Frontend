## Overview

The "Donation History" tab on the Donor Dashboard. Lists past fulfilled and expired pledges. Each row links back to the NGO profile for easy re-donation.

## Component Specification

### State (shared in DonorDashboard)

```js
const [history, setHistory] = useState([]);
const [historyLoading, setHistoryLoading] = useState(false);
```

### Fetch on Tab Switch

```js
useEffect(() => {
  if (activeTab !== "history") return;
  setHistoryLoading(true);
  api.get("/api/pledges/my/history")
    .then((res) => setHistory(res.data))
    .catch((err) => setError(err.response?.data?.message || "Failed to load history"))
    .finally(() => setHistoryLoading(false));
}, [activeTab]);
```

### Layout JSX

```jsx
{activeTab === "history" && (
  <div className="space-y-2">
    {history.length === 0 && (
      <p className="text-gray-400 text-center py-8">No donation history yet.</p>
    )}

    {history.map((item) => (
      <div
        key={item.pledgeId}
        onClick={() => navigate(`/ngo/${item.ngoId}`)}
        className="flex items-center gap-3 border rounded p-3 hover:bg-gray-50 cursor-pointer"
      >
        {/* Status icon */}
        <span className="text-lg">
          {item.status === "FULFILLED" ? "✅" : "⏰"}
        </span>

        {/* Category dot */}
        <span
          className="w-3 h-3 rounded-full"
          style={{ backgroundColor: CATEGORY_COLORS[item.category] }}
        />

        {/* Details */}
        <div className="flex-1">
          <p className="font-medium text-sm">{item.itemName} × {item.quantity}</p>
          <p className="text-xs text-gray-500">{item.ngoName}</p>
        </div>

        {/* Status + Date */}
        <div className="text-right">
          <span className={`text-xs px-2 py-0.5 rounded ${
            item.status === "FULFILLED"
              ? "bg-green-100 text-green-700"
              : "bg-gray-100 text-gray-500"
          }`}>
            {item.status === "FULFILLED" ? "Fulfilled" : "Expired"}
          </span>
          {item.fulfilledDate && (
            <p className="text-xs text-gray-400 mt-1">{item.fulfilledDate}</p>
          )}
        </div>
      </div>
    ))}
  </div>
)}
```

## Behaviour Notes

- Clicking a history row navigates to `/ngo/:ngoId` so the donor can re-donate
- Fulfilled items show ✅ with green badge; expired items show ⏰ with gray badge
- History is fetched lazily (only when the "Donation History" tab is active)
- No donor names are shown on the NGO side (privacy)
