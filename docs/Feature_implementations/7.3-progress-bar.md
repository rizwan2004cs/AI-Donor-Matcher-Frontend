# Feature 7.3 — Need Progress Bar

> **Branch:** `feat/progress-bar`

## Overview

A horizontal progress bar showing how much of an NGO's need has been pledged. Displays `pledged / required` amounts with a colour fill that grows from left to right. Turns green when fully met (100 %).

## Component Specification

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `pledged` | `number` | `0` | Quantity pledged so far |
| `required` | `number` | `1` | Total quantity needed |

### Full Component

```jsx
export default function NeedProgressBar({ pledged = 0, required = 1 }) {
  const pct = Math.min(Math.round((pledged / required) * 100), 100);
  const isFull = pct >= 100;

  return (
    <div className="w-full">
      {/* Label row */}
      <div className="flex justify-between text-xs text-gray-500 mb-1">
        <span>{pledged} / {required} pledged</span>
        <span>{pct}%</span>
      </div>

      {/* Bar */}
      <div className="w-full bg-gray-200 rounded-full h-2 overflow-hidden">
        <div
          className={`h-2 rounded-full transition-all duration-300 ${
            isFull ? "bg-green-500" : "bg-indigo-500"
          }`}
          style={{ width: `${pct}%` }}
        />
      </div>
    </div>
  );
}
```

## Behaviour Notes

- `pct` is clamped to 100 to prevent overflow visuals.
- `required` defaults to 1 to avoid division by zero.
- Transition on width change provides a smooth animation when data updates.
- Bar turns **green** at 100 % to give instant visual feedback.
