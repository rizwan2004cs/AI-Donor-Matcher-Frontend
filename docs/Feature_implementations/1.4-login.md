# 1.4 Login

Last updated: 2026-03-31

## Current Flow

1. User logs in with Firebase Email/Password.
2. Frontend reads Firebase ID token.
3. Frontend calls `POST /api/auth/firebase/login`.
4. Backend resolves user role and returns the app user payload.

## Redirect Rules

- donor -> `/`
- NGO -> `/ngo/dashboard` or `/ngo/complete-profile` based on profile state
- admin -> `/admin/dashboard`
