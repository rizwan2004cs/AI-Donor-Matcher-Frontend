## Overview

A higher-order route guard component that checks authentication and role before rendering child routes. Unauthenticated users are redirected to `/login`; users with the wrong role are redirected to `/`.

## Component Specification

```jsx
import { Navigate } from "react-router-dom";
import { useAuth } from "../auth/AuthContext";

export default function ProtectedRoute({ children, role }) {
  const { user } = useAuth();

  // Not logged in тЖТ redirect to login
  if (!user) return <Navigate to="/login" replace />;

  // Logged in but wrong role тЖТ redirect to home
  if (role && user.role !== role) return <Navigate to="/" replace />;

  // Authorised тЖТ render children
  return children;
}
```

### Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `children` | `ReactNode` | Yes | The page component to render |
| `role` | `string` | No | Required role (`"DONOR"`, `"NGO"`, `"ADMIN"`). If omitted, any authenticated user passes. |

## Usage in App.jsx

```jsx
{/* Donor only */}
<Route path="/pledge/:needId" element={
  <ProtectedRoute role="DONOR"><PledgeScreen /></ProtectedRoute>
} />
<Route path="/delivery/:pledgeId" element={
  <ProtectedRoute role="DONOR"><DeliveryView /></ProtectedRoute>
} />
<Route path="/dashboard/donor" element={
  <ProtectedRoute role="DONOR"><DonorDashboard /></ProtectedRoute>
} />

{/* NGO only */}
<Route path="/dashboard/ngo" element={
  <ProtectedRoute role="NGO"><NgoDashboard /></ProtectedRoute>
} />
<Route path="/dashboard/ngo/complete-profile" element={
  <ProtectedRoute role="NGO"><NgoProfileCompletion /></ProtectedRoute>
} />

{/* Admin only */}
<Route path="/dashboard/admin" element={
  <ProtectedRoute role="ADMIN"><AdminDashboard /></ProtectedRoute>
} />
```

## Edge Cases

1. **Token expired:** If a 401 is returned by the API, the Axios interceptor should call `logout()` to clear state and redirect. (This is an enhancement тАФ not strictly in ProtectedRoute but works in tandem.)
2. **Direct URL access:** User bookmarks `/dashboard/admin` тАФ ProtectedRoute checks on render, redirects if not admin.
3. **Role not provided:** If `role` prop is omitted, any logged-in user can access the route.
