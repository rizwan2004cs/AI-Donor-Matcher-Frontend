# 1.2 NGO Registration

Last updated: 2026-03-31

## Current Flow

1. NGO signs up with Firebase Email/Password in the frontend.
2. Frontend sends verification email through Firebase.
3. Frontend gets Firebase ID token.
4. Frontend calls `POST /api/auth/firebase/register`.
5. If NGO document upload is required, backend may accept multipart registration for NGO onboarding.

## Backend Ownership

- role = `NGO`
- NGO approval state
- profile completion state

## Important

Registration does not make the NGO publicly visible by itself. Discovery visibility depends on backend approval and valid map data.
