## Overview

Four stat cards at the top of the Admin Dashboard showing platform-wide metrics: Verified NGOs, Active Needs, Pledges Today, and Fulfilled This Month.

## API Contract

```
GET /api/admin/stats

Response (200):
{
  "verifiedNgos": 24,
  "activeNeeds": 87,
  "pledgesToday": 12,
  "fulfilledThisMonth": 156
}
```

---

## Component Specification

### State

```js
const [stats, setStats] = useState(null);
const [loading, setLoading] = useState(true);
```

### Fetch

```js
useEffect(() => {
  api.get("/api/admin/stats")
    .then((res) => setStats(res.data))
    .catch(() => {})
    .finally(() => setLoading(false));
}, []);
```

### Stats Strip JSX

```jsx
const STAT_CARDS = [
  { label: "Verified NGOs", key: "verifiedNgos", color: "bg-blue-50 text-blue-700" },
  { label: "Active Needs", key: "activeNeeds", color: "bg-green-50 text-green-700" },
  { label: "Pledges Today", key: "pledgesToday", color: "bg-orange-50 text-orange-700" },
  { label: "Fulfilled This Month", key: "fulfilledThisMonth", color: "bg-purple-50 text-purple-700" },
];

<div className="grid grid-cols-4 gap-4 p-6">
  {STAT_CARDS.map((card) => (
    <div key={card.key} className={`rounded-lg p-4 ${card.color}`}>
      <p className="text-3xl font-bold">{stats?.[card.key] ?? "—"}</p>
      <p className="text-sm mt-1">{card.label}</p>
    </div>
  ))}
</div>
```
