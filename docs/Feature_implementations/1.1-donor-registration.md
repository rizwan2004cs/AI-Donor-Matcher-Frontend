# 1.1 Donor Registration

Last updated: 2026-03-31

## Current Flow

1. Donor signs up with Firebase Email/Password in the frontend.
2. Frontend triggers Firebase verification email.
3. Frontend gets Firebase ID token.
4. Frontend calls `POST /api/auth/firebase/register`.

## Data Sent

```json
{
  "fullName": "Donor Name",
  "role": "DONOR",
  "location": "Nellore"
}
```

## Result

- donor app account is created in backend
- Firebase owns auth session
- backend owns donor role and app authorization
