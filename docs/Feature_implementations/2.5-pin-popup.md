# Feature 2.5 — Map Pin Popup

> **Branch:** `feat/pin-popup`
> **File:** `src/pages/DiscoveryMap.jsx` (inline) or `src/components/PinPopup.jsx`
> **Depends on:** DiscoveryMap (2.1), TrustBadge (7.2), CategoryPin (7.4)

---

## Overview

When a user clicks a map marker, a Leaflet Popup appears showing the NGO's photo, name, trust badge, distance, top need with remaining quantity, and a "View Full Profile" button.

## Component Specification

### Inline in DiscoveryMap Marker

```jsx
import { Marker, Popup } from "react-leaflet";
import { useNavigate } from "react-router-dom";
import { createCategoryIcon } from "../components/CategoryPin";
import TrustBadge from "../components/TrustBadge";
import { CATEGORY_COLORS } from "../utils/categoryColors";

const navigate = useNavigate();

{ngos.map((ngo) => (
  <Marker
    key={ngo.id}
    position={[ngo.lat, ngo.lng]}
    icon={createCategoryIcon(ngo.pinCategory)}
  >
    <Popup>
      <div className="w-56">
        {/* Photo + Name */}
        <div className="flex items-center gap-2 mb-2">
          {ngo.photoUrl ? (
            <img
              src={ngo.photoUrl}
              alt={ngo.name}
              className="w-10 h-10 rounded-full object-cover"
            />
          ) : (
            <div className="w-10 h-10 rounded-full bg-gray-200 flex items-center justify-center text-sm font-bold text-gray-500">
              {ngo.name.charAt(0)}
            </div>
          )}
          <div>
            <p className="font-bold text-sm">{ngo.name}</p>
            <TrustBadge score={ngo.trustScore} label={ngo.trustLabel} />
          </div>
        </div>

        {/* Distance */}
        <p className="text-xs text-gray-500">
          📍 {ngo.distanceKm.toFixed(1)} km away
        </p>

        {/* Top Need */}
        <div className="flex items-center gap-1.5 mt-1">
          <span
            className="w-2 h-2 rounded-full inline-block"
            style={{
              backgroundColor: CATEGORY_COLORS[ngo.pinCategory] || "#6B7280",
            }}
          />
          <span className="text-xs">
            <strong>{ngo.topNeedItem}</strong> — {ngo.topNeedRemaining} remaining
          </span>
        </div>

        {ngo.topNeedUrgency === "URGENT" && (
          <span className="inline-block mt-1 text-xs bg-red-100 text-red-600 px-1.5 py-0.5 rounded">
            Urgent
          </span>
        )}

        {/* CTA */}
        <button
          onClick={() => navigate(`/ngo/${ngo.id}`)}
          className="mt-2 w-full bg-blue-600 text-white text-xs py-1.5 rounded hover:bg-blue-700"
        >
          View Full Profile
        </button>
      </div>
    </Popup>
  </Marker>
))}
```

## Behaviour Notes

- **Photo fallback:** If `photoUrl` is null, show a circular div with the NGO's first initial
- **Trust badge:** Uses the shared `TrustBadge` component (see 7.2)
- **Category dot:** Matches the pin colour on the map
- **Urgency badge:** Only shown when `topNeedUrgency === "URGENT"`
- **View Full Profile:** Navigates to `/ngo/:id`
- Popup width is fixed at ~224px (`w-56`) so it doesn't overflow
