> **Branch:** `feat/admin-reports`

## Overview

A list of reported NGOs for admin review. Shows report count with a warning badge when ≥ 3 reports. Admin can expand to see individual reports, then dismiss them or suspend the NGO.

```
GET /api/admin/reports

    "ngoId": 3,
    "ngoName": "Shady Trust",

    "reports": [

```

### Dismiss Reports

```
DELETE /api/admin/reports/{ngoId}

Response (200):
{ "message": "Reports dismissed" }
```

### Suspend NGO

```
POST /api/admin/ngos/{id}/suspend

Response (200):
{ "message": "NGO suspended" }
```

## Component Specification

### State

```js
const [reports, setReports] = useState([]);
const [expandedNgo, setExpandedNgo] = useState(null); // ngoId to expand
```

### Layout JSX

```jsx
<div className="mt-8">
  <h2 className="text-lg font-semibold mb-3">Report Queue</h2>

  {reports.length === 0 && (
    <p className="text-gray-400 text-sm">No reported NGOs.</p>
  )}

  {reports.map((item) => (
    <div key={item.ngoId} className="border rounded mb-3">
      <div className="p-4 flex items-center justify-between">
        <div className="flex items-center gap-2">
          <span className="font-medium">{item.ngoName}</span>
          <span className="text-sm text-gray-500">{item.reportCount} reports</span>
          {item.reportCount >= 3 && (
            <span className="text-xs bg-yellow-100 text-yellow-700 px-2 py-0.5 rounded">
              ⚠️ High
            </span>
          )}
        </div>
        <div className="flex items-center gap-2">
          <button
            onClick={() => setExpandedNgo(
              expandedNgo === item.ngoId ? null : item.ngoId
            )}
            className="text-blue-600 text-sm underline"
          >
            {expandedNgo === item.ngoId ? "Collapse" : "View Reports"}
          </button>
          <button
            onClick={() => handleDismiss(item.ngoId)}
            className="bg-gray-200 px-3 py-1.5 rounded text-sm"
          >
            Dismiss
          </button>
          <button
            onClick={() => handleSuspend(item.ngoId)}
            className="bg-red-600 text-white px-3 py-1.5 rounded text-sm"
          >
            Suspend
          </button>
        </div>
      </div>

      {/* Expanded reports */}
      {expandedNgo === item.ngoId && (
        <div className="border-t px-4 py-3 bg-gray-50 space-y-2">
          {item.reports.map((r, i) => (
            <div key={i} className="flex items-center justify-between text-sm">
              <span className="font-medium">{r.reason}</span>
              <span className="text-gray-400">{timeAgo(r.reportedAt)}</span>
            </div>
          ))}
        </div>
      )}
    </div>
  ))}
</div>
```

### Handlers

```js
const handleDismiss = async (ngoId) => {
  try {
    await api.delete(`/api/admin/reports/${ngoId}`);
    setReports((prev) => prev.filter((r) => r.ngoId !== ngoId));
  } catch (err) {
    alert("Failed to dismiss reports");
  }
};

const handleSuspend = async (ngoId) => {
  if (!confirm("Suspend this NGO? This will cancel all their active needs and pledges.")) return;
  try {
    await api.post(`/api/admin/ngos/${ngoId}/suspend`);
    setReports((prev) => prev.filter((r) => r.ngoId !== ngoId));
  } catch (err) {
    alert("Failed to suspend NGO");
  }
};
```
